positionalColumnsCsv

  • positionalColumnsCsv(csv: string, fields: string[], __namedParameters?: { dynamicTyping?: boolean; header?: boolean; skipEmptyLines?: boolean }): any[]
    • Defined in src/core/csv.ts:73

    Takes a string with a CSV, parse it with Papaparse and build an object with column names based on position.

    For example, if the next CSV is given as input:

    a,b,c,d
0,1,2,3
4,5,6,7

    and the given fields are [ "x", "y" ] the output will be:

    [
      { x: 0, y: 1 },
      { x: 4, y: 5 }
    ]

    with [ "x", "y", "z", "t" ]:

    [
      { x: 0, y: 1, z: 2, t: 3 },
      { x: 4, y: 5, z: 6, t: 7 }
    ]

    with [ "x", "y", "z", "t", "u" ]:

    [
      { x: 0, y: 1, z: 2, t: 3, "u": undefined },
      { x: 4, y: 5, z: 6, t: 7, "u": undefined }
    ]

    Parameters

    • csv: string

      The CSV string to parse.

    • fields: string[]

      The fields to be transformed to by position.

    • __namedParameters: { dynamicTyping?: boolean; header?: boolean; skipEmptyLines?: boolean } = ...

      Papaparse configs, some of the most used ones. All true by default. Optional.

      • Optional dynamicTyping?: boolean
      • Optional header?: boolean
      • Optional skipEmptyLines?: boolean

    Returns any[]

    The transformed JSON object.
